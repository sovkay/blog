In June 2026, Brave shipped **Brave Origin**: a paid version of its browser with the crypto, AI, VPN, News, and the rest of the bolted-on junk taken out. Just the browser.

It costs **$59.99**, one-time. Free on Linux.

But Brave is open source, and Origin doesn't add anything. It ships the same browser with features turned off. You can do that yourself in about 30 seconds. No fork, no compiling, no trademark mess.

- - -

## What Origin actually is

Origin is the same Brave engine (Chromium + Brave's Shields ad/tracker blocking) with the monetization layers removed:

- Brave Rewards & Wallet (the crypto stuff)
- Leo, the built-in AI assistant
- Brave VPN, News, Talk
- Sponsored images, telemetry pings

None of that is *new code*. It's the standard browser with a set of switches flipped. On macOS, every one of those switches is an **enterprise policy**: a plist file the browser reads at launch and enforces, greying out the toggles so they stay off.

Origin sells you a set of defaults. So write the defaults yourself.

## The policy file

macOS reads managed browser policies from `/Library/Managed Preferences/com.brave.Browser.plist`. Drop this file there:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <!-- Web3 / crypto -->
  <key>BraveWalletDisabled</key><true/>
  <key>BraveRewardsDisabled</key><true/>

  <!-- AI -->
  <key>BraveAIChatEnabled</key><false/>

  <!-- Extras Origin strips -->
  <key>BraveVPNDisabled</key><true/>
  <key>BraveNewsDisabled</key><true/>
  <key>BraveTalkDisabled</key><true/>
  <key>TorDisabled</key><true/>
  <key>BravePlaylistEnabled</key><false/>
  <key>BraveSpeedreaderEnabled</key><false/>
  <key>BraveWaybackMachineEnabled</key><false/>
  <key>BraveWebDiscoveryEnabled</key><false/>

  <!-- Telemetry -->
  <key>BraveP3AEnabled</key><false/>
  <key>BraveStatsPingEnabled</key><false/>
  <key>MetricsReportingEnabled</key><false/>

  <!-- Uncomment for a truly bare install (these are real browser
       conveniences, not bloat — only turn them off if you mean it) -->
  <!--
  <key>PasswordManagerEnabled</key><false/>
  <key>AutofillAddressEnabled</key><false/>
  <key>AutofillCreditCardEnabled</key><false/>
  <key>TranslateEnabled</key><false/>
  -->
</dict>
</plist>
```

I deliberately left the *actual browser* features — password manager, autofill, translate, sync — switched on. Those aren't bloat, they're the browser. If you want a truly bare install, uncomment the block at the bottom of the file to switch those off too.

## Installing it

Save the file somewhere, then:

```bash
sudo mkdir -p "/Library/Managed Preferences"
sudo cp com.brave.Browser.plist "/Library/Managed Preferences/com.brave.Browser.plist"
```

Fully quit Brave (⌘Q — not just the window) and reopen it. Then visit `brave://policy`. Every key should show up as applied, and the matching settings will be locked and greyed out. That's it. That's Origin.

To undo, delete the file and flush macOS's preference cache, then restart Brave:

```bash
sudo rm "/Library/Managed Preferences/com.brave.Browser.plist"
sudo killall cfprefsd   # macOS caches managed prefs; deleting the file alone won't clear them
```

## Where it falls short

This isn't byte-for-byte identical to Origin:

- **It disables, it doesn't delete.** The crypto/AI/VPN code still ships inside the binary, just switched off. No disk or attack-surface savings. This is the one thing compiling from source would actually change, and it barely matters.
- **Cosmetic leftovers.** A disabled feature sometimes leaves a stray menu item or icon behind. The feature is off; the UI stub lingers.
- **Keys can silently no-op.** If Brave renames a policy key in an update, your entry does nothing and throws no error. Check `brave://policy` after big updates.
- **The toggles lock for you too.** Managed means managed. To flip one back you edit the plist and restart, not the settings UI. Same as Origin.

For a browser and nothing else, this gets you 95% of Origin. The missing 5% is dead code on disk and the odd menu stub. Cosmetic, not functional.

## Is it worth paying?

Sixty dollars buys you defaults someone else picked, a promise those features stay gone, and a way to support Brave. Reasonable if your time is worth more than the setup.

But if you'll paste a file into a folder, you already have what Origin sells. Save the file. Keep the money.
